myspacethe.net

What is it?

MySpaceThe.Net is a Google Chrome extension that brings all your memories into a reality! Words do not do justice to the beauty that is this chrome extension! To understand the beauty of the product we're talking about, you will just have to witness it with your own eyes!

How does it work?

We built this Google Chrome extension using the Clarifai API to gather tags from all the images of a single page and then feeding those tags to the Giphy API to find a suitable image to replace the previous image with. Pretty cool, right!?

How can I get it?

You can install this extension by following the link down below to the Google Chrome Extensions App Store. You can also find out more by visiting our GitHub repository.

What is it?

Not very fond of the command line? Can't remember those pesky git commands? This is your one-stop-shop to automating the deployment process for your website or web application!

How does it work?

This custom script is built on bash which is a command line scripting language. The script starts with a git commit, pushes it to the repository, then SSH's into your deployment server and pulls in any changes from the git repository.
